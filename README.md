# PopClip 扩展：中文标点规范化

一个 [PopClip](https://pilotmoon.com/popclip/) 扩展，用来一键把中文文本里**误用的英文（半角）标点、繁体角引号**，规范成简体中文的全角标点——尤其适合清理 AI（ChatGPT / Claude 等）输出的中文：它们常在中文里夹用英文逗号、括号、冒号和直引号，读起来很别扭。

转换是**单向**的（英 → 中），且**只在判定为中文语境时才动手**，因此不会破坏数字、网址、代码、时间、路径等内容。

## 效果

```
输入：他的命题是:商品交换在古老社会里就存在了,但只是偶然地出现(部落间的物物交换)。卢卡奇说「第二自然」(zweite Natur)其实是'人造世界'...对吗?

输出：他的命题是：商品交换在古老社会里就存在了，但只是偶然地出现（部落间的物物交换）。卢卡奇说“第二自然”（zweite Natur）其实是‘人造世界’……对吗？
```

## 转换规则

| 类别 | 半角 / 繁体 | → 全角（简体） | 说明 |
|---|---|---|---|
| 引号 | `"` `'` | `“ ”` `‘ ’` | 仅中文语境，自动成对；保护英文撇号 `don't` 与英寸 `6"` |
| 角引号 | `「」` `『』` | `“ ”` `‘ ’` | 繁体引号 → 简体引号 |
| 逗号 | `,` | `，` | |
| 分号 | `;` | `；` | |
| 冒号 | `:` | `：` | 后跟空格的“标签: 值”保持半角 |
| 问号 / 叹号 | `?` `!` | `？` `！` | |
| 句号 | `.` | `。` | 仅句末；小数 / 缩写 / 域名不动 |
| 括号 | `( )` | `（ ）` | 成对转换；函数调用与 Markdown 链接保持半角 |
| 省略号 | `...` | `……` | 仅贴着汉字时 |

## 不会误伤

判断逻辑是“标点紧贴汉字才转”，因此技术内容天然安全：

| 输入 | 输出（不变） | 原因 |
|---|---|---|
| `1,000,000` / `3.14` / `v2.0.1` | 同上 | 数字、版本号 |
| `12:30` / `2:1` | 同上 | 时间、比例 |
| `http://a.com/x?y=1` | 同上 | 网址 |
| `user@mail.com` | 同上 | 邮箱 |
| `C:/Users/me` | 同上 | 路径 |
| `func(x)` / `if(x>0)` / `[链接](http://x)` | 同上 | 代码、Markdown 链接 |
| `don't` / `6'5"` | 同上 | 英文撇号、英尺英寸 |
| `哈哈 :)` | 同上 | 颜文字 |
| `端口: 8080` | 同上 | 西式“标签: 值” |

Markdown 行内标记 `**粗体**`、`_斜体_`、`` `代码` ``、`~删除~` 会被「看穿」（不妨碍对汉字的判断），并原样保留。

## 安装

1. 在 Finder 里**双击** `中文标点规范化.popcliptxt`；
2. PopClip 弹窗询问是否安装，点确认即可；
3. 之后选中中文文本，点 PopClip 工具条上的“标”按钮，即把选区替换为规范化结果。

## 设计取舍

这些是有意为之、可按需在源码里微调的判断：

- **单向**：只做英 → 中，不会把中文标点转回英文（如需反向，见 git 历史里的旧扩展 `中英引号互转`）。
- **冒号后跟空格**（`命令: git push`）视为西式“标签: 值”，保持半角；中文行文里的冒号（`论证了:商品`）无空格，照常转全角。
- **落单括号**（无配对的 `(` 或 `)`）保持原样，以保护 `:)`、`[0,1)` 等。
- **句号**最保守：仅当其后不是字母 / 数字 / `/` / `@` 时才转 `。`。
- **幂等**：对已规范化的文本重复运行不会再改动，可放心多次触发。

`test-example.md` 是一份带 Markdown 粗体的中文样例，用于验证转换效果。
